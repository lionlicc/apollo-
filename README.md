# apollo-dbencrypt


## 以apollo-configservice为列
### 1、下载jasypt
https://mvnrepository.com/artifact/org.jasypt/jasypt/1.9.2

### 2、增加jasypt-spring-boot-starter依赖
 ```
 <dependency>
		<groupId>com.github.ulisesbocchio</groupId>
		<artifactId>jasypt-spring-boot-starter</artifactId>
		<version>1.7</version>
 </dependency>
 ```
